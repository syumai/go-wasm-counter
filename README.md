# go-wasm-counter

* Simple counter app written in go.
* This app running on browser.

## Example

https://syumai.github.io/go-wasm-counter/

## Environment

* go 1.11rc2

## Development

```console
go get go1.11rc2
go1.11rc2 download

make build
```

## How to run

```console
npm install -g serve
serve . // app served on localhost:5000
```
